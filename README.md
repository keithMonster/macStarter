# macStarter

macStarter 是一个用于快速启动 macOS 应用程序的工具。

构建一个 macOS 桌面端的应用启动器，替代或增强系统原生的 Launchpad。 核心价值：提供比原生 Launchpad 更快、更可定制、且风格更现代的应用管理界面。

## 功能

### 1 数据获取 (App Scanning)

这是应用的基础，必须能够自动发现系统中的应用。

扫描范围：

用户安装的应用

系统自带应用

当前用户的个人应用

(可选) (工具文件夹)

信息读取：

App 名称：读取 Info.plist 中的 CFBundleDisplayName 或文件名。

App 图标：直接通过 NSWorkspace 获取该文件的 NSImage。

App 路径：保存 URL 以便启动。

实时监听：

当用户安装新应用或删除应用时，列表应自动刷新（监听文件系统变更）。

### 2 视觉与交互 (UI/UX) - macOS 特性

macOS 拥有鼠标悬停、快捷键等独有交互。

激活方式：

全局快捷键：例如 Command + Command 唤起窗口。

菜单栏图标：点击顶部 Menu Bar 图标唤起。

窗口形态：

全屏覆盖 (Overlay)：类似原生 Launchpad，背景显示当前桌面壁纸的高斯模糊 (VisualEffectView - Material)。

网格交互：

Hover 效果：鼠标悬停在图标上时，图标放大 1.1 倍，或显示发光背景（仿 visionOS）。

点击启动：左键单击打开应用，窗口自动消失。

键盘导航：

支持方向键 ↑ ↓ ← → 选择图标。

支持 Enter 键启动当前选中的应用。

支持直接输入文字进行搜索（无需先点击搜索框）。

### 3 搜索模块 (Search)

即时过滤：用户打开界面后直接打字，网格内容实时过滤，只显示匹配的应用。

拼音支持：支持中文拼音首字母搜索（如输入 "wx" 显示 "微信"）。

### 4 文件夹

支持新建文件夹，文件夹可以包含多个应用，文件夹可以有名称和图标。

# 备注

它会在我自己的电脑上安装，不会上架到应用商店。

使用最新、ai 最友好的技术栈。
